Welcome to DaytaShare – Seize Your Dayta!

DaytaShare™ is a site I created to store and share my data. This site serves many purposes, but it started as a way for me to explore data visualization (specifically D3) as well as Python. I have a fitbit that I really enjoy using and decided to learn D3 using fitbit data (I use Python to capture the data).

Most recently, I've decided to use this as my own Python code repository. I tend to take many data science and machine learning courses, but the material from each only exists in it's own set of folders (ok, I know, I should probably take the time to get to know github a bit more). I also come across many needs for snippets and functions of code at work. By putting these pieces together here, it reinforces my knowledge as well as documents for future use.
